Github Org Browser
=======

- Inline React styles for CSS.
- JSX for templating.
- Express for server.

Running it locally
========
1. `git clone https://github.com/jamischarles/beastie.git`
2. `cd beastie`
3. `npm install`
4. `npm run start:dev`
5. Open `http://localhost:3000/` in your browser

Tested with Node 8.
